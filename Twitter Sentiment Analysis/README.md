# Twitter Sentiment Analysis

Performing sentiment analysis on tweets to classify them as positive or negative.

## Project Overview
This project focuses on analyzing public sentiment using the Sentiment140 dataset, which contains 1.6 million tweets. The goal is to build a robust model that can identify the emotional tone of a tweet.

## Key Components
- **Data Preprocessing**: Extensive text cleaning using Regular Expressions (Regex), stemming (PorterStemmer), and removal of stopwords.
- **Feature Extraction**: TF-IDF (Term Frequency-Inverse Document Frequency) vectorization.
- **Model**: Logistic Regression for binary classification.
- **Accuracy**: High performance achieved through proper text cleaning and feature engineering.

## Dataset
The dataset used is the Sentiment140 dataset, which includes fields like target, ids, date, flag, user, and text.

## How to Run
1. Open `Twitter_Sentiment_Analysis.ipynb`.
2. Follow the steps for data downloading and preprocessing.
3. Train and evaluate the model.
