# Titanic Survival Prediction

A classic machine learning project to predict the survival of passengers on the Titanic.

## Overview
This project explores the Titanic dataset to identify factors that influenced the survival of passengers. It implements a classification model to predict whether a passenger survived based on features like age, sex, passenger class, and fare.

## Workflow
1. **Exploratory Data Analysis (EDA)**: Understanding correlations between features and survival.
2. **Feature Engineering**: Handling missing values and encoding categorical variables.
3. **Modeling**: Training a classifier (e.g., Random Forest or Logistic Regression) on the passenger data.
4. **Evaluation**: Measuring model performance using accuracy and confusion matrices.

## Technologies Used
- **Python**
- **Pandas/NumPy** (Data Analysis)
- **Matplotlib/Seaborn** (Visualization)
- **Scikit-Learn** (Modeling)

## File Structure
- `prediction.ipynb`: Main notebook for analysis and modeling.
- `data/`: Contains `train.csv` and `test.csv` from Kaggle.

## Usage
Run the `prediction.ipynb` notebook in any Jupyter environment to see the analysis and training process.
