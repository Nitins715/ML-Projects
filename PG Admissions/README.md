# PG Admissions Prediction

Predicting the probability of a student getting admitted to a Post Graduate program based on various academic and research parameters.

## Project Overview
This project explores the factors that influence graduate admissions. It uses a regression approach to estimate the "Chance of Admit" based on several features.

## Dataset Features
- **GRE Score**: Graduate Record Examination scores.
- **TOEFL Score**: Test of English as a Foreign Language scores.
- **University Rating**: Rating of the undergraduate university.
- **SOP**: Statement of Purpose strength.
- **LOR**: Letter of Recommendation strength.
- **CGPA**: Undergraduate Grade Point Average.
- **Research**: Binary indicator (0 or 1) of research experience.

## Implementation
The analysis and modeling are performed in `predict.ipynb` using libraries like Pandas, NumPy, and Scikit-Learn.

## How to Run
1. Ensure you have the `data.csv` file in the same directory.
2. Open `predict.ipynb` and run the cells.
