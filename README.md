# Machine Learning Projects Collection

This repository contains a collection of Machine Learning projects covering various domains including Computer Vision, Natural Language Processing, and Predictive Analytics.

## Projects List

### 🟢 Beginner Level
1. **[Book Recommendation System](./Book%20Recommendation)**: A Flask-based web application that provides book recommendations using popularity-based and content-based approaches.
2. **[Car Price Prediction](./Car%20Price%20Prediction)**: A regression-based project to predict used car prices using a Linear Regression model with data from Quikr.
3. **[Dogs vs Cats Classification](./DogsvsCats)**: A Deep Learning project using Convolutional Neural Networks (CNN) to classify images of pets.
4. **[Laptop Price Predictor](./Laptop%20Price%20Predictor)**: A Streamlit application that estimates laptop prices based on user-defined hardware specifications.
5. **[Titanic Survival Prediction](./Titanic%20Survival)**: A classification project predicting passenger survival on the Titanic, exploring key demographic factors.
6. **[MNIST Handwritten Digit Classification](./MNIST%20Dl)**: A Deep Learning project using TensorFlow and Keras to classify handwritten digits from the MNIST dataset.
7. **[Movie Recommender System](./Movie%20Recommendation)**: A Streamlit-based application that suggests similar movies using content-based filtering and fetches posters via API.
8. **[PG Admissions Prediction](./PG%20Admissions)**: A regression analysis project to estimate the probability of graduate admission based on academic profiles.
9. **[SMS Spam Classifier](./SMS%20Spam%20Classifier)**: An NLP project that classifies SMS messages as Spam or Ham using TF-IDF and a trained classification model.
10. **[Twitter Sentiment Analysis](./Twitter%20Sentiment%20Analysis)**: A sentiment analysis project that classifies tweets as positive or negative using Logistic Regression and NLP techniques.

### 🟡 Medium Level
11. **[Game Of Thrones Personality Matcher](./2%20-%20(Med)%20Level/GOT(T-Sne))**: A Streamlit-based Web Application that helps you discover similar characters from the Game of Thrones universe using t-SNE.
12. **[IPL Win Predictor](./2%20-%20(Med)%20Level/Ipl%20Winning%20Probability)**: A Machine Learning based web application that predicts the winning probability of an ongoing IPL cricket match.
13. **[Olympics Data Analysis](./2%20-%20(Med)%20Level/Olympics%20Data%20Analysis)**: An interactive web application built with Streamlit that provides a comprehensive and visual analysis of historical Olympic data.
14. **[T20 Cricket Score Predictor](./2%20-%20(Med)%20Level/T20%20score%20predictor)**: A machine learning based web application that predicts the final score of a T20 cricket match in the first innings.
15. **[WhatsApp Chat Analyzer](./2%20-%20(Med)%20Level/Whatsapp%20Chat%20Analysis)**: A Python-based Streamlit web application that analyzes exported WhatsApp chat text files and provides statistical insights.
